2. Тематика сайта, название:
Тематика:
Интерактивная платформа для покупки билетов на культурные и развлекательные мероприятия (концерты, театры, спортивные события, выставки, кино и тп).

Название сайта:
«Фишка»

__________________________________________________________________________________________________

3. Цель и задачи системы:
Цель:
Разработать веб-приложение, позволяющее пользователям удобно покупать билеты на мероприятия, оставлять отзывы и управлять своим профилем с помощью системы аутентификации.

Задачи:
    🔸 Спроектировать базу данных (мероприятия, пользователи, заказы, отзывы и др).
🔸 Реализовать регистрацию / авторизацию с разграничением ролей (пользователь, администратор).
🔸 Создать каталог мероприятий с фильтрацией (по дате, категории, цене, городу и другим параметрам).
🔸 Разработать систему бронирования и оплаты билетов (не уверены насчёт оплаты).
🔸 Добавить функционал отзывов и оценок (только для купивших билет или с спец пометкой).
🔸 Реализовать личный кабинет с историей покупок и написанными отзывами.

__________________________________________________________________________________________________

4. Целевая аудитория:
Молодёжь (18–30 лет): активные посетители концертов, стендапов, фестивалей.

Администраторы площадок / ивент-менеджеры (малая доля, для добавления событий).      < ------- В перспективе развития проекта

5. Содержание сайта:
   
Страницы и разделы:
Главная страница — афиша, подборка популярных мероприятий, поиск.
Каталог мероприятий — карточки событий с фильтрами и сортировкой.
Страница мероприятия — описание, дата, место, цена, доступные места (схема зала), кнопка «Купить», блок отзывов.
Корзина и оформление заказа — выбранные билеты, итоговая сумма, форма оплаты (демо).

Личный кабинет:
Мои билеты (активные и прошедшие)
История покупок
Мои отзывы (возможность редактирования/удаления)
Настройки профиля
Форма регистрации / входа (JWT-токены, сессии).

Админ-панель — добавление/редактирование мероприятий, управление пользователями.

__________________________________________________________________________________________________

6. Функциональные возможности:
Для всех пользователей (без авторизации):
Просмотр каталога и страниц мероприятий
Поиск и фильтрация
Чтение отзывов

Для авторизованных пользователей:
Покупка билетов (выбор мест, количество, демо-оплата)
Добавление отзыва и оценки (только если билет куплен и мероприятие прошло)
Редактирование / удаление своих отзывов
История покупок и электронные билеты в ЛК
Смена пароля и аватара

Для администраторов:
CRUD операций с мероприятиями (добавление, удаление, перенос даты)
Просмотр всех заказов
Удаление нежелательных отзывов
Блокировка / разблокировка пользователей

Технические функции:
Аутентификация (email + пароль) с валидацией
Защита от SQL-инъекций и XSS
Адаптивный дизайн (для мобильных устройств)
Пагинация списка мероприятий
Валидация форм на клиенте и сервере
