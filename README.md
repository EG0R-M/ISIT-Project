О проекте

«Фишка» — это интерактивная платформа, объединяющая покупателей, организаторов мероприятий и администраторов. Проект решает проблему разрозненности афиш и отсутствия проверенной обратной связи.

## 2. Тематика и название

Тематика:  
Интерактивная платформа для покупки билетов на культурные и развлекательные мероприятия (концерты, театры, спортивные события, выставки, кино и т.п.).

Название сайта: «Фишка»

## 3. Цель и задачи системы

Цель:
Разработать веб-приложение, позволяющее пользователям удобно покупать билеты на мероприятия, оставлять отзывы и управлять своим профилем, а организаторам — создавать собственные события и отслеживать продажи.

Задачи:
1 Спроектировать базу данных (мероприятия, пользователи, заказы, отзывы, залы, места, билеты, платежи и др.).
2 Реализовать регистрацию / авторизацию с разграничением ролей (пользователь, организатор, администратор).
3 Создать каталог мероприятий с фильтрацией (по категориям, дате, поиску).
4 Разработать систему бронирования и оплаты билетов (выбор мест, схема зала, демо-оплата, чек).
5 Добавить функциональность отзывов и оценок (только для купивших билет и посетивших мероприятие).
6 Реализовать личный кабинет с историей покупок, избранным и написанными отзывами.
7 Предоставить организаторам конструктор мероприятий (создание площадок, залов, сеансов).
8 Реализовать админ-панель для модерации событий и управления пользователями.
9 Разработать REST API для внешних интеграций.
10 Интегрировать сервисы погоды (OpenWeatherMap) и карт (Яндекс.Карты).
11 Обеспечить адаптивный дизайн
12 Тестирование

## 5. Содержание сайта

Страницы и разделы:
* Главная страница — афиша, карусель популярных площадок, виджет погоды, кнопки категорий.
* Каталог мероприятий — карточки событий с фильтрацией по категориям и поиском.
* Страница мероприятия — описание, дата, место, цена, доступные сеансы, схема зала, блок отзывов.
* Бронирование и оплата — выбор мест, временная блокировка на 15 минут, демо-оплата, чек.

Личный кабинет:
* Мои билеты (активные, ожидающие оплаты, прошедшие)
* История покупок
* Мои отзывы (редактирование / удаление)
* Избранное
* Настройки профиля (смена пароля, телефона, имени)

Для организаторов (дополнительно):
* Мои мероприятия (список со статистикой продаж)
* Создание мероприятия (конструктор площадки, зала, сеансов)

Админ-панель:
* Модерация мероприятий (одобрение / отклонение)
* Управление мероприятиями (CRUD)
* Управление пользователями (блокировка / разблокировка)
* Статистика платформы

## 6. Функциональные возможности

Для всех пользователей (без авторизации):
- Просмотр каталога и страниц мероприятий
- Поиск и фильтрация
- Чтение отзывов
- Просмотр погоды (виджет и страница с анимациями)

Для авторизованных пользователей:
- Покупка билетов (выбор мест, схема зала, демо-оплата)
- Добавление отзыва и оценки (только если билет куплен и мероприятие прошло)
- Редактирование / удаление своих отзывов
- История покупок и электронные билеты в ЛК
- Смена пароля и редактирование профиля
- Добавление мероприятий в избранное

Для организаторов:
- Все возможности обычного пользователя
- Создание мероприятий (площадка, зал, сеансы)
- Просмотр статистики продаж по своим событиям
- Управление созданными мероприятиями

Для администраторов:
- Модерация мероприятий (одобрение / отклонение)
- CRUD операций с любыми мероприятиями
- Просмотр всех заказов
- Удаление нежелательных отзывов
- Блокировка / разблокировка пользователей

API (REST):
- Получение списка событий, сеансов, доступных мест
- Создание и отмена бронирования
- Имитация оплаты
- Получение и добавление отзывов
- Получение данных о погоде и популярных площадках

Технические функции:
- Аутентификация (email + пароль) с хешированием
- Защита от SQL-инъекций (ORM) и XSS (Jinja2)
- CSRF-защита форм
- Адаптивный дизайн (мобильные устройства)
- Пагинация списка мероприятий (12 карточек)
- Валидация форм на клиенте и сервере
- Переключение тёмной / светлой темы с сохранением в localStorage
- Интеграция с OpenWeatherMap API (погода)
- Интеграция с Яндекс.Картами (геокодирование адресов)

## 7. Технологический стек

Backend:
- Python 3.13
- Flask (микрофреймворк)
- SQLAlchemy (ORM)
- SQLite (разработка) / PostgreSQL (продакшен)
- Flask-Login (аутентификация)
- Werkzeug (хеширование паролей)
- Requests (внешние API)

Frontend:
- HTML / CSS (кастомные стили, тёмная/светлая тема)
- JavaScript (Vanilla)
- Bootstrap 5 (адаптивная сетка, компоненты)
- Jinja2 (шаблонизация)

Инструменты:
- Git / GitHub (контроль версий)
- GitHub Projects (планирование задач)
- Discord / Telegram (коммуникация)
